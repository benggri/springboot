# Spring Boot Sample Project

## 프로젝트 간략 소개

- Spring Boot 를 사용한 간단한 API 커뮤니티(게시판) 서비스
- Front 서비스(예정)
  - 사용자 : React or Vue
  - 관리자 : Spring Boot Thymeleaf

## 사용 기술

```
현재까지 사용한 기술(업데이트 중)
```

- java 11
- Spring Boot 2.7.10
- Spring Security
- JWT
- MyBatis
- JPA
- MariaDB
- thymeleaf
- bootstrap 5
- jquery
- axios
